# 选择排序
- N个数字，排序
| 范围 | 操作                     | 操作次数            |
| ---- | ------------------------ | ------------------- |
| 0-N  | 遍历，与记录范围内最小的 | N遍历，N比较，1swap |
| 1-N  |                          |                     |    |                          |                     |
- **复杂度：时间复杂度O($N^2$)      空间复杂度：O(1)  有限的中间变量**

## 冒泡排序
- 从0开始，相邻的位置上，谁大谁向右移
- 从1开始.....
- **复杂度：时间复杂度O($N^2$)      空间复杂度：O(1)  有限的中间变量**

## 插入排序
- 0 1 小的放到前面
-  2 放到不必它小的后面
-  3 放到不比他小的后面
- **复杂度：时间：倒序情况O($N^2$) 正序O(N)   ** 

## 异或运算
- 相同为0； 不同为1
- 可以理解为无进位相加
- 0 ^ N = N;    N^ N = 0
- 满足交换律、结合律    **和某一位上某个出现的次数有关，但是与出现的位置无关**
- 交换a, b
```java
a = a^b;
b = a^b;
a = a^b;
```
